# Audience (default)

> Embedded default. Override at `docs/marketing/audience.md`.

## Primary persona

- Role: software / product builder using the product hands-on
- Pain today: a manual, repetitive task that drains focus
- Success metric: hours saved per week, friction removed

## Triggers

- Just finished a feature and needs to announce it
- Sees a competitor ship a similar feature without clear differentiation
- Receives user feedback that the product "does more than they knew"

## Copy implications

- Lead with the pain they already feel, not the feature name.
- Use the reader's vocabulary, not the internal project vocabulary.
