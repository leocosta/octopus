# Brand (default)

> This is an embedded default. Override it by creating
> `docs/marketing/brand.md` in the target repository.

## Identity

- Product name: `{{PRODUCT_NAME}}`
- One-line pitch: `{{PRODUCT_PITCH}}`
- Primary audience: see `audience.md`

## Visual

- Color palette: neutral (grays) + single accent color; avoid rainbow gradients.
- Typography: clean sans-serif for body; bolder sans-serif for headlines.
- Logo: use the repository logo if present under `public/` or `images/`;
  otherwise omit and let the post rely on typography.

## Do / Don't

- Do favor specific, concrete phrasing.
- Do keep visual noise low — whitespace is a feature.
- Don't use stock-photo business clichés.
- Don't inflate outcomes or invent numbers.
