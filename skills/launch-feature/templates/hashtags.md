# Hashtags (default)

> Embedded default. Override at `docs/marketing/hashtags.md`.

## Rules

- Keep hashtag count modest per channel.
  - Instagram: 5–12
  - LinkedIn: 3–5
  - X: 1–3 inline
- Mix: 1 brand tag + 2–3 category tags + 1–2 long-tail tags.
- Avoid exhausted generic tags ("#innovation", "#technology").

## Default tag pool

Use only as fallback when the repo provides no override:
`#BuildInPublic #ShipFast #ProductLaunch #IndieMakers #DevProductivity`

## Per-post selection

Pick 3–5 tags from the pool that match the post's angle. When in doubt,
prefer specificity over reach.
