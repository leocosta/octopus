# Social Media Guide (default)

> Embedded default. Override at `docs/marketing/social-media-guide.md`.

## Cadence

- 1 launch post per channel per feature.
- Reuse the email body for a long-form LinkedIn version.
- Stagger posts: IG first, LI same day, X thread next day.

## Channel fit

- **Instagram** — emotional hook + short value bullets + CTA. Visual-first.
- **LinkedIn** — context + mechanism + outcome. Longer form (1000–1500 chars).
- **X** — thread of 5–8 tweets. Hook tweet must work standalone.
- **Email** — subject that earns the open; body that earns the click.
- **LP** — headline + subheadline + 3 bullets; no fluff.

## Approval gate

Nothing is posted without explicit human review. Launch-kit files are drafts
until the reviewer signs off.
