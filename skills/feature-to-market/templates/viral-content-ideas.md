# Viral Content Ideas (default)

> Embedded default. Override at `docs/marketing/viral-content-ideas.md`.

## Angles that travel

- Side-by-side comparison (us vs. manual process)
- Time-lapse / before-after screenshot
- Behind-the-scenes build note
- Contrarian take on a common practice
- Unexpected use case from a real user

## Pitfalls

- Posting a feature description without a reason to care.
- Using AI-style filler: "In today's fast-paced world…"
- Claiming "best-in-class" without evidence.
