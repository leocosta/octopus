## How to Use

1. **Before a task**: Read this index, then load the relevant domain's `rules.md` and `hypotheses.md`
2. **During a task**: If a hypothesis is being tested, note the outcome
3. **After a task**: Update `knowledge.md` with new facts, update `hypotheses.md` if any observation needs validation
