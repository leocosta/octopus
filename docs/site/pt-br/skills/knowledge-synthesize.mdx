Uma base de conhecimento vale mais que a soma das notas só quando algo a
*atravessa* e revela um link que sua memória perdeu. Por si só, toda nota é uma
ilha. `knowledge-synthesize` encontra as conexões que cruzam nós — o mesmo
blocker atingindo duas áreas, uma nota que contradiz a decisão que cita, o
write-up antigo que de repente voltou a ser relevante.

## O que resolve

O custo de uma base em silos é invisível: você só paga quando *não* conectou duas
coisas e entregou a decisão pior por causa disso. Um padrão cross-node nunca
aparece a menos que você já desconfie dele e vá procurar — o que significa que na
maioria das vezes não aparece. Esta skill faz a base procurar por você.

## Como resolve

O core determinístico (`octopus synthesize`) computa **candidatos** de conexão
ranqueados sobre o registry `octopus kr`; a skill faz os julgamentos que um
script não faz.

O core revela três sinais:

- **shared-target** — dois nós que linkam o mesmo terceiro (se relacionam por ele);
- **co-mention** — uma entidade que recorre em vários nós mas não tem nó próprio
  (um tópico que merece ser promovido);
- **relevant** — com `--node`, outros nós ranqueados por overlap de entidades com
  um nó-foco: a busca do *esquecido-mas-relevante*.

A extração de entidades é **neutra de idioma** — lê `[[wikilinks]]` e spans de
`` `code` ``, que funcionam em qualquer língua, e deixa a detecção de nomes
próprios em texto livre pra skill, em vez de uma heurística fixa em inglês.

A skill então julga quais candidatos são reais, lê um nó contra as autoridades
que ele linka (um ADR, uma spec) pra sinalizar **contradições**, e — com `--fix`
— semeia um link só quando uma menção resolve pra exatamente um nó, deixando os
casos ambíguos só como relatório.

## Uso & parâmetros

```
octopus synthesize [--root <id>] [--node <path>] [--fix]
```

| Flag | O que faz | Default |
|---|---|---|
| `--root <id>` | Sintetiza sobre um knowledge root. | todo root resolvido |
| `--node <path>` | Busca esquecido-mas-relevante pra um nó (ranqueia outros por entidades compartilhadas). | varredura do root inteiro |
| `--fix` | Semeia um link quando a menção resolve pra exatamente um nó (reversível). | só relatório |
